# P.20220709 蚂蚁过河
# v1

有一天，下起了大雨，把一群蚂蚁的家淹了。于是，它们决定搬家。

但它们家门口流着一条小河，如果想搬家，就得过这条河。它们找了一片叶子当木筏。几只蚂蚁爬上木筏，剩下的蚂蚁尝试把木筏推过河。结果，木筏不但没推过去，反而把木筏卡在了河中间。那些在木筏上的蚂蚁搭成了一座蚂蚁桥，让一只蚂蚁爬过桥到岸上寻找木屑当桨。突然，那只蚂蚁打滑了，掉进了水里。最靠近的那只蚂蚁抓住它的手，把它拉了上来。接着，又有一只蚂蚁去找木屑了。那只蚂蚁把木屑运回了船上，所有的蚂蚁就回到了船上，开始划船。当木筏漂到了对岸，一只蚂蚁待在了船上，把木筏划了回去。最后，蚂蚁们用同样的方式全过了河。

现在，蚂蚁就可以继续搬家了。
